# BirdBudz Dashboard

BirdBudz Dashboard is a public frontend prototype for visualizing bird-health monitoring alerts, image uploads, and environmental signals.

This version uses simulated data only. It does not connect to private APIs, production systems, or real-time monitoring infrastructure.

## What it shows

- Alert cards
- Monitoring dashboard layout
- Simulated bird-health screening results
- Human-review workflow concept
- Frontend design for an AI biosurveillance platform

## What is not included

This repo does not include backend code, private datasets, production APIs, model weights, or deployment credentials.

## Tech stack

React · TypeScript · Tailwind CSS · Vite

## Status

Educational prototype.
